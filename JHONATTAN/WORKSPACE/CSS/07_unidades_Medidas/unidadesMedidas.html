Unidades de medida em CSS

  • Unidades de medida absolutas
  • Unidades de medida relativas

Unidades de medida absolutas

As unidades de medida absolutas são fixas e não dependem do tamanho da fonte ou do elemento pai.

  • px: pixel
  • cm: centímetro
  • in: polegada

Unidades de medida relativas

As unidades de medida relativas são proporcionais ao tamanho da fonte ou do elemento pai.

  • em: tamanho da fonte do elemento pai
  • rem: tamanho da fonte do elemento raiz
  • %: porcentagem do tamanho do elemento pai
